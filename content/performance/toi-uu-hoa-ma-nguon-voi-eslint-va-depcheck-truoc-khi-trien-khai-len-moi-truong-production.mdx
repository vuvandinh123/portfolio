Khi phát triển ứng dụng web, việc duy trì chất lượng mã nguồn là rất quan trọng. Hai công cụ mạnh mẽ giúp bạn làm điều này là ESLint và Depcheck. Dưới đây là cách sử dụng chúng để làm sạch mã nguồn của bạn.
### 1. ESLint Kiểm tra mã nguồn
ESLint là một công cụ phân tích mã tĩnh giúp bạn tìm kiếm các lỗi cú pháp và các vấn đề về chất lượng mã trong JavaScript. Dưới đây là các bước để thiết lập và sử dụng ESLint:
1. Cài đặt ESLint:
Để cài đặt ESLint, bạn chỉ cần chạy lệnh sau trong thư mục dự án của mình:
```bash
npm install eslint --save-dev
```
Để sử dụng ESLint với React, bạn cần cài đặt thêm các package sau:
```bash
npm install eslint-plugin-react --save-dev
```
> Lưu ý: Nếu bạn sử dụng TypeScript, bạn cần cài đặt thêm `@typescript-eslint/parser` và `@typescript-eslint/eslint-plugin`.

Nếu bạn muốn sử dụng eslint toàn cầu, bạn cần cài đặt nó toàn cầu bằng câu lệnh sau:
```bash
npm install -g eslint
```

2. Tạo file cấu hình:
```bash
npx eslint --init
```
Sau khi cài đặt, bạn cần tạo một file cấu hình `.eslintrc` trong thư mục gốc của dự án. Dưới đây là một số cấu hình cơ bản:
```json
{
  "extends": "eslint:recommended",
  "rules": {
    "no-console": "off"
  }
}
```
3. Kiểm tra mã nguồn:
Sử dụng ESLint để kiểm tra 1 file mã nguồn của bạn bằng cách chạy lệnh sau:
```bash
npx eslint yourfile.js
```
Nếu bạn muốn kiểm tra toàn bộ dự án, bạn có thể chạy lệnh sau:
```bash
npx eslint . --fix
```
Lệnh trên sẽ kiểm tra toàn bộ dự án và tự động sửa các lỗi cú pháp nếu có thể.
### 2. Depcheck Kiểm tra các module không sử dụng
Depcheck là một công cụ giúp bạn kiểm tra các module không sử dụng trong dự án của mình. Dưới đây là cách sử dụng Depcheck:
1. Cài đặt Depcheck:
Bạn cài đặt Depcheck toàn cầu bằng câu lệnh sau:
```bash
npm install -g depcheck
```
Nếu bạn muốn cài đặt Depcheck local, bạn có thể chạy lệnh sau:
```bash
npm install depcheck --save-dev
```

2. Kiểm tra các module không sử dụng:
Khi depcheck được cài đặt ở local chạy lệnh sau:
```bash
npx depcheck
```
Và khi depcheck được cài đặt toàn cầu chạy lệnh sau:
```bash
depcheck
```
Kết quả sẽ hiển thị tất cả các module không sử dụng trong dự án của bạn. Bạn có thể xóa chúng bằng cách chạy lệnh sau:
```bash
npm uninstall <module_name>
```
